## Задача

Признаки до начала игры:

- Разница рейтингов
- Цвет фигур

Признаки в некоторый момент игры:

- Номер хода
- Мои фигуры на доске
- Преимущество по фигурам
- Оставшееся у меня время
- Преимущество по времени
